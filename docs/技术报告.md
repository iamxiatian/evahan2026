# 技术报告

Qwen2.5-VL 是 Qwen 视觉语言系列的最新旗舰模型，在基础能力与创新功能上均有显著提升。该模型通过增强的视觉识别、精准目标定位、强大文档解析和长视频理解能力，实现了对世界更深入的理解与交互。

Qwen2.5-VL能够通过边界框或点准确定位物体；从发票、表格等文档中提取结构化信息，并支持图表、图示与版面分析；采用动态分辨率处理与绝对时间编码技术，可处理不同尺寸的图像和长达数小时的视频，实现秒级事件定位，从而在不依赖传统归一化方法的情况下，直接感知空间尺度与时间动态。

Qwen2.5-VL模型通过从头训练原生动态分辨率视觉 Transformer（ViT）并结合窗口注意力机制，在保持原生分辨率的同时降低计算开销。因此，Qwen2.5-VL 不仅在静态图像与文档理解方面表现优异，还能作为交互式视觉代理，在操作电脑、手机等实际场景中进行推理、工具调用与任务执行。
